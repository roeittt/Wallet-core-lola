# Product Overview

**LOLA WALLET** is a comprehensive wallet development repository containing multiple blockchain wallet implementations and SDKs.

## Core Components

- **Rabby Wallet**: Open-source browser extension for DeFi ecosystem providing secure multi-chain experience
- **Ethers.js**: Complete Ethereum library for dapps, wallets and blockchain tools
- **OKX Web3 JS Wallet SDK**: Multi-chain TypeScript/JavaScript SDK for offline transaction signing
- **Trust Wallet Core**: Cross-platform, mobile-focused cryptographic wallet library supporting 130+ blockchains
- **Solana Web3.js**: JavaScript SDK for Solana blockchain interactions

## Target Use Cases

- Browser extension wallet development
- Multi-chain wallet applications
- DeFi protocol integrations
- Mobile wallet implementations
- Offline transaction signing
- Cross-platform blockchain development

## Key Features

- Multi-chain support (Ethereum, Bitcoin, Solana, Cosmos, and 130+ others)
- Secure key management and transaction signing
- Browser extension architecture
- Mobile-first design patterns
- Offline transaction capabilities
- Hardware wallet integrations
